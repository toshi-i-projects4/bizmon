この画面でできること

本登録済みユーザーの情報を個別に編集します。招待中・初回登録待ちの方はこの画面には表示しません。

・この画面では users に登録済みのユーザーのみを表示します。 ・編集できる項目は、社員コード、氏名、部署、役職、ビズもん権限、状態、申込者、備考です。 ・メールアドレス、企業ID、企業名、登録UIDは内部情報として扱い、この画面には表示しません。 ・この画面では Firebase Authentication 側のメールアドレス・パスワード変更やAuth削除は行いません。

本登録済みユーザーの検索

自社 users を検索し、選択して編集します。完全削除ではなく無効化を基本にします。
